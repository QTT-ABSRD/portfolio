# Portfolio

Projet portfolio avec Next.js et architecture MVC.

## Structure du projet

Le projet suit une architecture MVC (Model-View-Controller) :

- **models/** : Modèles de données et logique métier
- **controllers/** : Contrôleurs qui gèrent la logique de contrôle
- **views/** : Composants React (vues)
- **app/** : Routes Next.js (App Router)

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur.

## Build

```bash
npm run build
npm start
```
